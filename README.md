写来给自己用的ASS导入DSC工具，因为我找不到能导入歌词颜色的工具

部分资料参考了nasty的Open-PD-Script-Editor:https://github.com/nastys/Open-PD-Script-Editor

如果你不知道怎么写ASS文件，可以使用aegisub制作

自动生成歌词部分的pv_db

支持导入颜色标签，但颜色标签必须添加在歌词最开头
